Learn how to use basic Python programming to interact with ChatGPT and other large language models. This hands-on workshop walks through the fundamentals of calling LLM APIs — from setting up your environment and sending prompts, to structuring outputs and building simple AI-powered scripts. No prior LLM experience needed.
